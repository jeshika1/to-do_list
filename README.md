# to-do_list
This is a simpl to-do_list application built using python's turtle module. It allows the user to add, delete, edit, and clear all task using keyboard shortcut visually on a graphics screen.
Features:
Addtask: Press A to add the task on the list
Clear task: Press C to clear all the tasks from the list
Delete task: Press D to delete a particular task that user wants
Edit task: Press E to edit a task
